Proyectos


Mr. Bigote

Programa de asistencia personal y entretenimiento. Un programa hecho con Windows forms que permite mantenr un control de víveres y cuando se terminan, sistema de avisos en forma de pop-ups, poder escuchar la radio, testear conexiones TCP y generar contraseñas seguras.

ERP talleres

ERP de gestión para talleres pequeños. Es un programa que está pensado para empresas talleres con un solo terminal, ya que guarda los datos de forma local y codificados en formato XML. El programa permite mantener un registro de Clientes, vehículos, tareas, presupuestos y facturas.

Esta aplicación aún se encuentra en proceso de desarrollo.

Librería PDF

Permite crear documentos de forma sencilla y con conocimientos de programación mínimos.
